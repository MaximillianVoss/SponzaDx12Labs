# Лабораторная работа 4

Четвёртая работа посвящена оптимизации сцены при большом количестве объектов.

Что делалось:

- добавлено размещение большого количества объектов в сцене;
- реализовано отсечение объектов по пирамиде видимости камеры (`frustum culling`);
- добавлены переключатели режимов для сравнения поведения с отсечением и без него;
- реализована пространственная структура данных для ускорения отбора видимых объектов;
- подготовлен вариант использования `octree` для разбиения сцены.

Результат:

Приложение получило базовые средства оптимизации рендера для сложной сцены с большим числом объектов.
